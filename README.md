# 接口反射的实现及使用

**作业：**

有一个接口切片：

```go
type TSlice struct{
    i []interface{}
    //你需要的其他字段
    type	type	//添加字段type，当队列存入第一个元素时，通过typeof确定该队列的元素类型。
}

t:=TSlice{}
t.push()
t.pop()
t.top()
t.length()
```

需要实现：

- 实现队列，先入先出

- push方法（入一个），pop方法（出一个），top方法（返回队头元素），length方法（返回长度）

- 切片的元素实例可以是任意类型，但同一个TSlice实例的元素只能是一个类型
